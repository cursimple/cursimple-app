# 独立考试提醒规则设计

## 目标

新增独立的考试提醒规则/设置项，默认提前 40 分钟；规则自动覆盖课表中的全部考试，并允许用户临时取消某一场考试的提醒。考试日期过去后，临时取消记录自动清理。

## 规则模型

在 `ReminderScopeType` 中新增 `Exam`。它与 `SingleCourse` 不同，不绑定单个 `courseId`，而是匹配当前课表里 `category == Exam` 的所有 `CourseItem`。

`ReminderRule` 新增 `mutedCourseIds` 字段，默认空列表。该字段用于 `Exam` 规则的单场考试排除：如果某场考试 ID 在列表中，考试仍展示在课表里，但全局考试提醒不会为它生成闹钟。

## 规划与同步

`ReminderPlanner` 展开 `Exam` 规则时只匹配考试，并跳过 `mutedCourseIds` 中的考试。触发时间仍由考试开始时间减去 `advanceMinutes` 计算，默认值由 UI 和 ViewModel 使用 40 分钟。

考试日期过去后，在提醒同步前清理 `Exam` 规则中的过期排除项：如果某个被排除考试已经没有今天及未来的有效发生日期，就从 `mutedCourseIds` 中移除并保存规则。清理只更新规则，不删除考试，也不影响用户单独创建的 `SingleCourse` 考试提醒。

## 界面

提醒设置中新增“考试提醒”设置项，可开启/关闭、设置提前分钟数、选择铃声。设置项保存为 `Exam` 规则。

考试详情或考试卡片操作中新增临时取消/恢复入口：

- 如果全局考试提醒未开启，提示先开启考试提醒。
- 如果已开启且当前考试未被排除，显示“取消本场考试提醒”。
- 如果当前考试已被排除，显示“恢复本场考试提醒”。

提醒规则列表显示“考试提醒”，说明为“自动提醒全部考试”，并追加“已临时取消 N 场”。

## 测试

- `ReminderPlannerTest` 覆盖 `Exam` 规则只匹配考试，不匹配普通课程。
- 覆盖 `mutedCourseIds` 会跳过单场考试。
- 覆盖过期考试排除项会在同步前清理，未来考试排除项保留。
- 运行相关单测与 `assembleDebug`。
